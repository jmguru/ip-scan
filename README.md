# ip-scan
Simple ip-scanner

## Usage

**ip-scan**

  Simply scans a network given an ip and port.

**Options**

  --**ip** _string_ : IP address or range like 192.168.1.0/24

  --**port** _string_ : Port or port range like 21-23,80

## Example

`node ip-scan.js --ip '192.168.1.0/24' --port '21-23,80'`
